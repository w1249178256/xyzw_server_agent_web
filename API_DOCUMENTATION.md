## 需要的接口清单

### 1. 用户认证模块

#### 用户注册
- **路径**: 待确认
- **方法**: POST
- **请求参数**:
  - username: 用户名
  - password: 密码
  - email: 邮箱（可选）
- **响应**: 用户信息 + token

#### 用户登录
- **路径**: 待确认
- **方法**: POST
- **请求参数**:
  - username: 用户名
  - password: 密码
- **响应**: 用户信息 + token

#### 获取当前用户信息
- **路径**: 待确认
- **方法**: GET
- **响应**: 当前用户详细信息

#### 用户登出
- **路径**: 待确认
- **方法**: POST
- **响应**: 成功/失败状态

### 2. 游戏账号管理模块

#### 获取游戏账号列表
- **路径**: 待确认
- **方法**: GET
- **查询参数**:
  - page: 页码
  - pageSize: 每页数量
- **响应**: 游戏账号列表（分页）

#### 绑定游戏账号
- **路径**: 待确认
- **方法**: POST
- **请求参数**:
  - gameName: 游戏名称
  - gameAccount: 游戏账号
  - gamePassword: 游戏密码（加密）
  - serverName: 服务器名称（可选）
  - remark: 备注（可选）
- **响应**: 绑定成功的账号信息

#### 更新游戏账号
- **路径**: 待确认
- **方法**: PUT
- **请求参数**:
  - id: 账号ID
  - 其他可更新字段
- **响应**: 更新后的账号信息

#### 删除游戏账号
- **路径**: 待确认
- **方法**: DELETE
- **路径参数**: accountId
- **响应**: 删除成功/失败

#### 获取游戏账号详情
- **路径**: 待确认
- **方法**: GET
- **路径参数**: accountId
- **响应**: 账号详细信息

---

### 3. 任务管理模块

#### 获取任务列表
- **路径**: 待确认
- **方法**: GET
- **查询参数**:
  - page: 页码
  - pageSize: 每页数量
  - status: 任务状态（待执行、执行中、已完成、失败等）
  - gameAccountId: 游戏账号ID（可选）
- **响应**: 任务列表（分页）

#### 创建任务
- **路径**: 待确认
- **方法**: POST
- **请求参数**:
  - taskName: 任务名称
  - taskType: 任务类型
  - gameAccountId: 关联的游戏账号ID
  - config: 任务配置（JSON）
  - scheduledTime: 计划执行时间（可选）
- **响应**: 创建的任务信息

#### 获取任务详情
- **路径**: 待确认
- **方法**: GET
- **路径参数**: taskId
- **响应**: 任务详细信息

#### 更新任务
- **路径**: 待确认
- **方法**: PUT
- **请求参数**:
  - id: 任务ID
  - 其他可更新字段
- **响应**: 更新后的任务信息

#### 删除任务
- **路径**: 待确认
- **方法**: DELETE
- **路径参数**: taskId
- **响应**: 删除成功/失败

#### 启动/停止任务
- **路径**: 待确认
- **方法**: POST
- **请求参数**:
  - taskId: 任务ID
  - action: start/stop
- **响应**: 操作结果

---

### 4. 任务执行状况监控模块

#### 获取任务执行记录
- **路径**: 待确认
- **方法**: GET
- **查询参数**:

  - page: 页码
  - pageSize: 每页数量
  - startTime: 开始时间

- **响应**: 执行记录列表（分页）

#### 获取任务执行详情
- **路径**: 待确认
- **方法**: GET
- **路径参数**: executionId
- **响应**: 执行详情（包括日志、状态、耗时等）

#### 获取实时任务状态
- **路径**: 待确认（可能需要 WebSocket）
- **方法**: GET / WebSocket
- **响应**: 当前所有运行中任务的实时状态

#### 获取任务统计数据
- **路径**: 待确认
- **方法**: GET
- **查询参数**:
  - timeRange: 时间范围（今天、本周、本月等）
- **响应**: 统计数据（成功率、执行次数、平均耗时等）

## 接口规范说明

### 通用响应格式
```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

### 认证方式
- 使用 JWT Token
- Header: `Authorization: Bearer <token>`

### 错误码
- 200: 成功
- 400: 请求参数错误
- 401: 未认证
- 403: 无权限
- 404: 资源不存在
- 500: 服务器错误
